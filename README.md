# FinGraph Analytics 📈

Trabalho final da disciplina **FGA0030 - Estruturas de Dados 2 (EDA2)**.

## 🎯 Objetivo Principal
Desenvolver um sistema de análise textual focado no **Mercado Financeiro**, utilizando **Grafos de Coocorrência**. O sistema mapeia relações entre termos financeiros em documentos, identificando conexões, tendências e padrões de relevância semântica.
